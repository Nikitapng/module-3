Описание проекта

Создание страницы каталога для сайта компании “Интерьер”. На странице представлены:
	•	Карточки товаров с интерактивными элементами.
	•	Текстовое поле для подписки на новости компании.

Используемые технологии
	•	Figma — для проектирования макета страницы.
	•	VSCode — для написания и редактирования кода.
	•	HTML — структура страницы.
	•	CSS — стилизация и интерактивность элементов.
